Welcome to sitey-buildey!

You can write both markdown and HTML files in the source/ directory, and they will be built into a website in the site/ directory.

Lines of plain text in markdown files will be converted to <p> elements, and a title beginning with # on the first line will be processed into the index.

Titles in HTML pages should follow the format <center><h1>:: Your Title Here ::</h1></center> in order to be recognized by the script (if you change it, just make sure to update the associated lines in config.sh).

You can choose whether to use a pre-formatted navigation on the left or the automatically assembled one. In either case, index.html will be automatically built by the script to include all pages.
